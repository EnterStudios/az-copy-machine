API Token Instructions

  1. In AgileZen, go to the Settings page and Developer tab.
  2. Click the "New API Key" button.
  3. Add a Description (e.g., AZ Copy Machine) and click "Save Changes".
  4. Copy the API Token below and click "Load API Access".
  5. You must have API access to all Projects you wish to copy to and from.
