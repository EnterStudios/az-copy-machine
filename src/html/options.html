API Token Instructions

  1. In AgileZen, go to the Settings page.
  2. Switch to the Developer tab.
  3. Click the "New API Key" button.
  4. Add a Description (e.g., AZ Copy Machine) and make sure to click "Save Changes".
  5. Copy the API Token below and save.
  6. You must have Read+Write API access to all Projects you wish to copy to and from.
